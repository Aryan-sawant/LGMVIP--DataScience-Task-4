# Image to Pencil Sketch Conversion

![Pencil Sketch Conversion Demo]

## Overview

Welcome to the Image to Pencil Sketch Conversion project! This repository contains the code and resources for transforming images into stunning pencil sketches using the OpenCV library. The artistic process involves a sequence of steps, including gray scaling, inverting, blurring, and inverting the blurred image.

## Installation

To get started with the project, follow these installation steps:

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Aryan-sawant/LGMVIP--DataScience-Task-4.git
   ```

2. Install the required libraries using pip:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

- Run the pencil sketch conversion:
   ```bash
   python LGM_Task4.ipynb sketch.jpg
   ```

- Experience the magic of transforming images into captivating pencil sketches!

Our project delivers stunning pencil sketches by leveraging OpenCV and a series of artistic transformations. Explore the artistic side of image processing!
